## magento2的docker-compose环境
### 使用方法
#### clone magento2环境
```
git clone https://github.com/kk0501/magento2-docker.git
```

#### 复制代码
```
cd magento2-docker/src
git clone https://github.com/kk0501/magento2.git
```

#### 启动项目
```
接上一步， docker回到根目录
cd ../
sudo docker-compose up -d
```

### 访问
浏览器中输入地址进行安装
